## 1. 문제 12 : n^2이 3의 배수이면 n은 3의 배수임을 증명하라.

- 대우로 증명 : n이 3의 배수가 아니라면 n^2은 3의 배수가 아니다.

  1. n = 3k + 1인경우

  (3k + 1)^2 = 9k^2 + 6k + 1 = 3(3k^2 + 2k) + 1
  3의 배수가 아니다.

  2. n = 3k + 2인경우

  (3k + 2)^2 = 9k2 + 12k + 4 = 3(3k^2 + 4k + 1) + 1
  3의 배수가 아니다.
